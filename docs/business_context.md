# Contexto de negocio

Una compañía telco gestiona sus emplazamientos de red a través de tres módulos operativos:

- **Centros TX**: equipos de transmisión y transporte.
- **Sitios RAN**: sitios de acceso radio (antenas, celdas, etc.).
- **Infraestructura**: torres, mástiles y otros elementos físicos donde se montan equipos.[file:21]

Cada módulo se ha gestionado históricamente con su propio sistema y catálogo de emplazamientos, lo que genera:

- Duplicidades y códigos no alineados.
- Dificultad para tener una visión única de la red.
- Problemas para medir disponibilidad y evolución de la infraestructura.

Este proyecto unifica los tres mundos en un modelo único de emplazamientos para facilitar el análisis operativo y la construcción de dashboards de seguimiento.
